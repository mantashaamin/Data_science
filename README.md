# 📊 Data Science Learning Journey

This repository documents my journey of learning Data Science from the fundamentals to Machine Learning through hands-on coding projects and notebooks.

Rather than only collecting code, this repository focuses on understanding concepts, implementing them from scratch, and solving small exploratory data analysis (EDA) problems.

## 🎯 Learning Approach

For every topic, I try to follow the same workflow:

```
Concept
    ↓
Prediction
    ↓
Implementation
    ↓
Visualization
    ↓
Explanation
```

This helps me understand **why** a technique is used instead of simply memorizing syntax.
